NON-SQL User

The textbox you are presented with at the main page can be treated as an SQL text editor. If you do not know how to write an SQL query, this may not be the best website for you. If you are one of those people, try inputting the following into the input box:

select State, Total_Votes, Democratic_Percent, Republican_Percent from Previous_Elections where Year=1972

As you may have noticed, the map changes colors according to your inputted year and hovering over states gives its respective State, Democratic Percent, Republican Percent, Independent Percent, and Other Percent for that respective year. If you want further use from this website, you need to understand how to write simple queries using SQL

SQL User

States that would normally have a space between them such as New York are instead called NewYork without the space, thus if you want to run a query with a state condition, it must follow this format:

select Year, State, Republican_Percent from Previous_Elections where State="NewYork"

If you do no use Year as a condition, the frontend map will default to the year 2016, this is purely for visuals ans will not affect your resultant table

All correct syntax attributes are listed below:

Table: Demographics

State | White_Percent | Black_Percent | Hispanic_Percent | Asian_Percent | AmericanIndian_Percent | NativeHawaiin_Percent | With_Children_Percent | Without_Children_Percent | Male_Percent | Female_Percent | Median_Income | Citizen_Percent | Non_Citizen_Percent | AdultsUnder55 | AdultsOver55 | FullTime_Percent | PartTime_Percent | Unemployed_Percent | Education_Rank | its primary key is {State}

Table: Previous_Elections

State | Total_Votes | Democratic_Percent | Republican_Percent | Independent_Percent | Other_Percent | Votes_Democratic | Votes_Republican | Votes_Other | Year | its primary key is {State, Year}

Table: Candidates

Name | Vice_President_Candidate | Party | Popular_Vote | Electoral_Votes | Year | its primary key is {Name, Year}

Relation: Winners {Candidates, Previous_Elections}(Join on Year)

Made by running: select c.name, c.Vice_President_Candidate, c.Party, c.Popular_Vote, p.Year from Previous_Elections p JOIN Candidates c ON p.Year = c.Year Where Electoral_Votes>270 GROUP BY p.Year

Relation: Polls {Demographics, Previous_Elections}(Join on State)
